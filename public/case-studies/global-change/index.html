Functional Mini-Site Prototype for Global Change

Created a deployable, high-fidelity prototype using reusable components, brand styling, and headless CMS integration.

Overview

Developed a functional mini-site prototype using reusable components from an internal design system. Integrated a headless CMS to simulate dynamic content and deployed the project as a working site for client review. The prototype demonstrated real interactions, branding, and content behavior beyond static mockups.

My Role

  • Wrote HTML/CSS/JS prototypes using components from the internal USWDS-based library
  • Applied brand-specific styling to create high-fidelity interactive frames
  • Connected Jekyll to Contentful for dynamic content management
  • Deployed the prototype on Netlify
  • Delivered a functional mini-site demonstrating accurate interactions, branding, and content

Technical Approach

The prototype was built using the existing internal component library to ensure consistency with production systems. Components were extended with brand-specific styling while preserving their accessibility and structural patterns.

Content was delivered dynamically by connecting Jekyll templates to Contentful entries, allowing the prototype to simulate a real content workflow rather than relying on static placeholder content.

The site was deployed to Netlify for easy stakeholder access, enabling the client team to explore the prototype as a working site and provide feedback on navigation, layout, and interactions.
